# Contributing to TraceAgent

Thank you for your interest in contributing to TraceAgent.

## How to contribute

1. Fork the repository.
2. Create a feature branch.
3. Make your changes and test them locally.
4. Open a pull request with a clear summary.

## Guidelines

- Keep changes focused and easy to review.
- Do not commit secrets, credentials, or private environment values.
- Update documentation when behavior or setup changes.
- Prefer small, well-tested PRs.

## Development checklist

- Backend runs locally with the provided environment template.
- Frontend builds and starts without environment errors.
- Documentation reflects any new setup steps.
